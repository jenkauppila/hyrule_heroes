# Hyrule Heroes

## Status: Archived

This was a project from Module 2 (Advanced Python) of my Coding Temple bootcamp, one month in. It's kept here for reference but is no longer maintained or updated.

## What This Was
A turn-based CLI game inspired by The Legend of Zelda: Breath of the Wild. Players choose one of four champions, each with unique stats and two special abilities, and battle Ganon in a turn-based combat loop.

## Structure
- `main.py`: entry point and game flow
- `characters.py`: base `Character` class plus subclasses for each playable champion and the boss
- `create_character.py`: character selection menu
- `battle.py`: turn-based combat loop

## Tech Used
Python (standard library only), object-oriented design with class inheritance

## Why It's Archived
This was a bootcamp exercise, not a production project. It doesn't reflect my current skill level or coding standards, but it's a solid early example of applying OOP concepts. Kept for personal reference only.
